# placedog.js
 placedog.js utilizes the [place.dog API](https://place.dog) to download placeholder images of dogs

# Installation
```bash
npm i placedog.js
```

# Usage
### Javascript
```javascript
const placedog = require("placedog.js");
placedog(1920, 1080, "path/to/file.jpg");
```
### Typescript
```typescript
import placedog from "placedog.js"
placedog(1920, 1080, "path/to/file.jpg");
```
